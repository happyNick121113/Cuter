📦 File Splitter

GitHub 文件分割工具 - 突破 25MB 限制

💡 使用说明

  • 选择要分割的文件（支持任意类型）
  • 设置每个分块的大小（默认 24MB，适合 GitHub）
  • 点击"开始分割"按钮
  • 所有分块会打包下载为 ZIP 文件
